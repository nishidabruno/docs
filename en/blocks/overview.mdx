Before diving into the details, let’s understand how **Canvasit** works.\
Image creation in Canvasit is powered by node-based **blocks**. These blocks are interconnected and enhanced by AI models, which generate text prompts and images.

In the following sections, you’ll learn more about how blocks work and explore all the different types you can use.
